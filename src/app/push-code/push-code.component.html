Push Code to GitHub

To push code to GitHub, follow these steps:

  1. Initialize a local Git repository:
    2. git init
  2. Add your files to the repository:
    3. git add .
  3. Commit your changes with a meaningful message:
    4. git commit -m "Initial commit"
  4. Create a new repository on GitHub:
    5. git remote add origin <repository_url>
git push -u origin master
  5. Enter your GitHub credentials (username and password) when prompted.
  6. Your code will be pushed to the remote repository on GitHub.
